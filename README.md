# WIFI_Connect
For Company WIFI Groups which will use built-in password.
请大家使用ASUS_WIFI (Version 1.1)

相对于前一版，有显著更新。

说明：
1.	支持四种wifi加密方式 
2.	将隐藏的wifi: ATHZ-TEST0571(15S) 和ATHZ-TEST0571(15S) 自动添加在wifi列表中
3.	定制CJ86GJI4 代理

【用途】
集合了公司內部以的一些wifi，內置wifi密碼（有   標示的）。
限於其他組的wifi隱私問題，請勿傳播。
